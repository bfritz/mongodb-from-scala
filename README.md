## Using MongoDB from Scala - Simple Demo Code

Bootstrapping:

    git clone git@github.com:bfritz/mongodb-from-scala.git
    cd mongodb-from-scala
    sbt update
    sbt ~console

From the sbt console:

    new casbah.BeerDemo
